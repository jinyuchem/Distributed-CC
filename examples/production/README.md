# Production Examples

This directory is a placeholder for production-scale examples that will be
added later.

Until full production templates are added, see
[`../../docs/parameters.md`](../../docs/parameters.md) for the parameter and
logging flag quick reference.

Planned contents include:

- Slurm job scripts for multi-node runs
- recommended environment setup and `pytblis` build notes
- parameter templates for RCCSDT, RCCSDT(Q), and RCCSDTQ
- restart and output-directory conventions
- logging settings for memory and timing for communication and contractions
